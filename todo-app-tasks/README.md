# Todo App Tasks — สร้างแอปทีละขั้นตอน

## ภาพรวม
Task files เหล่านี้แบ่งการสร้าง Todo App ออกเป็น 6 ขั้นตอน
เหมาะสำหรับเรียนรู้การใช้ GitHub Copilot ทีละส่วน

## ลำดับการทำ

| Task | หัวข้อ | สิ่งที่ได้ |
|------|--------|----------|
| 1 | โครงสร้างโปรเจค | folders + package.json |
| 2 | Server Setup | Express + middleware |
| 3 | REST API | CRUD endpoints |
| 4 | React Setup | CRA + App.js |
| 5 | TodoForm | ฟอร์มเพิ่ม todo |
| 6 | TodoList + TodoItem | แสดง/แก้/ลบ todo |

## วิธีใช้กับ Copilot
1. เปิดไฟล์ task ในแต่ละขั้นตอน
2. ใช้ Copilot Chat อ้างอิงไฟล์ task เป็น context
3. หรือใช้ Reusable Prompt ที่เกี่ยวข้อง (เช่น `/setup-project`, `/implement-api`)
